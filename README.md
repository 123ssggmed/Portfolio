# Santiago Garces Chavarria Portfolio

This repository hosts the public professional portfolio for Santiago Garces Chavarria.

The curated web portfolio is published from `docs/` and is designed for two audiences:

- Recruiters evaluating remote roles in research, strategy, fintech, AI workflow execution, and consulting support.
- Clients evaluating project support in strategy research, pricing, embedded finance, diligence, and AI-enabled operations.

## Public Site

Open the portfolio site from GitHub Pages after Pages is enabled for:

- Source: `main`
- Folder: `/docs`

Expected URL:

`https://123ssggmed.github.io/Portfolio/`

## Portfolio Structure

- `docs/index.html` - public portfolio homepage
- `docs/styles.css` - visual system for the portfolio site
- `_archive/README.md` - archive policy for legacy or non-public material

The homepage links only to the strongest public-facing PDFs. Older documents, prompts, logs, raw prospect lists, drafts, and internal operating notes are intentionally excluded from the public navigation.

If Pages is not active, open the repository settings and enable:

4. Branch: `main`
5. Folder: `/docs`
